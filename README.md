Sim2TerrainAI: Off-Road Semantic Segmentation

## 🚀 Overview

**Sim2TerrainAI** is an advanced AI-powered platform for **Off-Road Semantic Segmentation**, designed to enable autonomous systems to understand and navigate complex natural environments. Leveraging state-of-the-art Deep Learning models, the system processes synthetic terrain data to provide high-fidelity scene understanding.

The project utilizes **Synthetic Data generated by Falcon Duality AI**, providing a diverse and safely-simulated environment for training robust terrain analysis models.

## 🛠️ Tech Stack

- **Deep Learning**: PyTorch, Segmentation Models PyTorch (SMP), Albumentations
- **Model Architecture**: DeepLabV3+ with ResNet50 Encoder (Pre-trained on ImageNet)
- **Backend**: FastAPI (Python)
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Framer Motion
- **Inference**: GPU-accelerated (CUDA support)

## 📊 Dataset Insight

The model is trained on synthetic datasets provided by **Falcon Duality AI**. This synthetic data allows for perfect ground truth masks and exposure to extreme terrain conditions that are difficult to capture in the real world.

## 🧠 Model Architecture

The core of Sim2TerrainAI is a **DeepLabV3+** model. This architecture is chosen for its ability to capture multi-scale context through Atrous Spatial Pyramid Pooling (ASPP), which is critical for identifying terrain features of varying sizes.

- **Encoder**: ResNet50 (Pre-trained on ImageNet)
- **Decoder**: Refines segmentation boundaries using low-level features.
- **Input Resolution**: 512x512 pixels.
- **Loss Function**: Combined Dice + Cross-Entropy loss for handling class imbalance.

## 📈 Performance

The model achieves high mIoU (Mean Intersection over Union) on synthetic test sets, specifically excelling in distinguishing between different types of low-lying vegetation and rocks, which are crucial for off-road path planning.
